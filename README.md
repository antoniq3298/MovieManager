Movie Manager is a simple web application for managing personal movie collections. Users can add, view, and organize their favorite movies.

## Features

- Add a new movie with title, genre, and year.
- View a list of your movies.
- User authentication: login and registration.
- Beautiful responsive UI with fixed-size buttons and elegant layout.

## Technologies Used

- HTML5
- CSS3
- PHP
- Git for version control

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/MovieManager.git

Open the project in VS Code.

Make sure you have a local web server (e.g., XAMPP or WAMP) running PHP.

Place the project in the htdocs (or similar) folder and access via browser:
http://localhost/MovieManager/
Usage

Register a new user or log in if you already have an account.

Add movies using the "Add Movie" form.

View or delete your movies in the "My Movies" section.
